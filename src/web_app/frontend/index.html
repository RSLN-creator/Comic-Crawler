功能说明

实现一键转化为CBZ格式，方便阅读。CBZ是漫画阅读器（如Kavita、Komga、Calibre）广泛支持的格式， 将下载的漫画图片打包成单个文件，便于管理和阅读。

使用步骤

  1. 1. 选择包含专辑子目录的源目录
  2. 2. 选择CBZ文件输出目录
  3. 3. 点击"开始打包"按钮
  4. 4. 等待打包完成后导入Kavita
