## 🚀 新特性

### ✨ 核心功能
- **完整的 Web UI** - Kavita 风格的现代化界面
- **漫画下载** - 批量下载，支持 JM 前缀 ID
- **CBZ 打包** - 一键转换，完美兼容 Kavita/Komga
- **设置持久化** - 自动保存用户配置
- **实时日志** - 下载/打包进度实时显示
- **终端输出** - 实时显示 Python 终端内容

### 🔧 服务管理
- **一键启动** - start.bat 自动启动前后端（隐藏窗口）
- **一键停止** - end.bat 或 Web UI 红色按钮
- **端口配置** - 默认后端 8765，前端 3000

## 🚀 快速开始

### Windows 用户

1. 下载并解压 `Comic-Crawler-v1.0.0-source.zip`
2. 双击运行 `start.bat`
3. 浏览器自动打开 http://localhost:3000
4. 开始使用！

### 停止服务
- 方法 1：在 Web UI 设置页面点击红色"停止服务"按钮
- 方法 2：双击运行 `end.bat`
